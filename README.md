## 소규모 전자상거래 플랫폼

여러 판매자가 입점하여 상품을 판매하고, 고객은 다양한 상품을 구매할 수 있는 B2B2C 형태의 전자상거래 플랫폼입니다.

- **인증/인가**: JWT + Naver OAuth
- **사용자 구분**: USER, SELLER, ADMIN
- **상품 관리**: 카테고리, 상품 CRUD, 재고 관리, 판매자 대시보드
- **주문 관리**: 장바구니, 주문, 결제, 배송 추적
- **리뷰 시스템**: 상품 리뷰 CRUD
- **쿠폰 시스템**: 쿠폰 발급 (웰컴/생일/VIP 자동 발급)
- **알림 시스템**: 재입고/발주/쿠폰 알림

---

### 역할 구분

| 역할 | 권한 |
|------|------|
| **ROLE_USER** | 주문, 리뷰, 재입고 투표/알림, 쿠폰 사용 |
| **ROLE_SELLER** | USER 권한 + 상품 관리, 재고 관리, 판매 대시보드 |
| **ROLE_ADMIN** | SELLER 권한 + 회원 관리, 카테고리 관리, 쿠폰 관리 |

---

## 기능

### 1. 재입고 관리 시스템

- **재입고 투표 (동기)**: 사용자가 품절 상품에 투표
  - Redis 기반 실시간 투표 수 집계

- **자동 알림 (비동기)**: 재입고 알림 신청 사용자에게 알림 발송
  - 재고 트랜잭션과 분리된 비동기 이벤트 처리

- **Event-Driven**: 상품 재고 변경 시 Spring Event 발행
  - 재입고 후 투표 초기화와 알림 발송을 이벤트 리스너에서 수행

### 2. 자동 발주 권장 (스케줄러)

- **매일 00:00 실행**: 재고 부족 상품 감지

- **평균 판매량 기반**: 지난 7일 평균 × 안전 계수

- **발주 권장 수량 계산**: 목표 재고 - 현재 재고

### 3. 쿠폰 발급

- **웰컴 쿠폰**: 회원가입 시 자동 발급

- **생일 쿠폰**: 매일 자동 스케줄러로 생일자에게 발급

- **VIP 쿠폰**: 누적 구매금액 기준 자동 발급

- **쿠폰 검증**: 최소 주문금액, 만료일, 사용 여부 체크
